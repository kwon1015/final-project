질문 : 뜨거운 물이 들어있는 냄비에 나무국자와 금속 국자를 넣었는데, 나무국자는 뜨거워지지않고 금속국자는 뜨거워지는 이유는 무엇일까요?	
키워드 : 열, 전달, 재질, 나무, 금속	
모범답안 : 나무 재질보다 금속 재질이 열 전달이 더 잘 되기 때문이다.	

q2
질문 : 볼록 렌즈를 이용하여 사물을 보면 평면 유리로 보는 것에 비하여 어떻게 다르게 보일까요?	
키워드 : 확대, 거꾸로	
모범답안 :  평면유리로 보는 것보다 사물이 더 확대되어 보이고, 먼 거리에 있는 것을 바라보면 사물이 거꾸로 보인다.

q3
질문 : 여러 제과점이 서로 경쟁을 하면 소비자에게 어떤 점이 좋을까요?	
키워드 : 가격, 품질, 다양성, 혜택	
모범답안 :  제품의 가격이 낮아지고, 품질이 좋아진다. 또 제품의 다양성이 증가하고, 소비자들은 더 좋은 혜택을 받을 수 있다
